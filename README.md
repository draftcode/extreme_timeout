# ExtremeTimeout

Do it in 40 seconds, or die.

    require 'extreme_timeout'
    ExtremeTimeout::timeout(40) do
      do_something
    end

## Installation

Add this line to your application's Gemfile:

    gem 'extreme_timeout'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install extreme_timeout
